Простой хлопковый выключатель сетевых (220V) ламп, на базе ATtiny85.
Использует амплитудный анализ звука и при правильной настройке позволяет
практически избежать случайных срабатываний на посторонние звуки.

Проект на EasyEDA: https://oshwlab.com/nich1con/clap-kit
